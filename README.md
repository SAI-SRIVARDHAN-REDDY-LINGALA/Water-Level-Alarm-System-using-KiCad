# Water Level Alarm System using KiCad

## Overview

This project implements a simple and practical **Water Level Alarm System** designed using **KiCad PCB Design Suite**.

The system detects the presence of water using conductive sensing probes and activates an alarm or buzzer when water reaches a predefined level.

The project focuses on:

- Basic electronic sensing principles
- PCB schematic design
- PCB layout design
- Practical embedded hardware thinking
- Signal detection circuits
- Real-world safety and monitoring applications

The complete design was developed using:

- KiCad Schematic Editor
- KiCad PCB Layout Designer

# Working Principle

Water conducts electricity due to dissolved ions.

The system uses conductive probes placed at different water levels.

When water touches the sensing probes:

```text
Electrical path completes
        ↓
Detection circuit activates
        ↓
Alarm/Buzzer turns ON
```

Water Probes
